How to Use

Quick guide to the Equipment Explorer

Search

Find cables, equipment, and documents by searching for their tag numbers or filenames.

Search Types:

  • All Tags - Search for cables and equipment by exact tag number or description
  • Cables - Filter results to only show cables
  • Equipment - Filter results to only show equipment
  • Documents - Search for a PDF by filename, supplier or supplier code to see all tags it contains

Tips:

  • Enable Partial match to find tags containing your search term
  • Click on any tag in results to search for it
  • Use the Contents button on PDF results to see all tags in that document
  • Pin the cable, equipment or PDF to the sidebar to easily find it later

Cable List

Browse all cables with their connection information.

  • View start and destination equipment for each cable
  • See compartment locations and deck information
  • Click any cable or equipment tag to search for it
  • Use the search box to filter the table
  • Export to CSV for use in Excel
  • Pin the cable to the sidebar

Documents

Browse all drawings and documents.

Status Indicators:

  • Indexed - Document has been processed and tags can be searched
  • OCR Done - OCR processing complete
  • Needs OCR - Document needs OCR processing
  • Not Indexed - Document not yet in the index

Actions:

  • Open - Opens the PDF in a new tab
  • Contents - Shows all cables and equipment found in the document
  • Tags count - Click to see a popup with all tags in the document
  • Pin - Pin the PDF to the sidebar

Offline Mode

The application caches data locally for offline use.

  • Cable and document lists are cached in your browser
  • Data automatically syncs when you're back online
  • Use the Refresh button to manually update cached data
  • The cache indicator shows when you're using cached data
